\subsection{Detailed Description}
This struct implement an \hyperlink{structIPv4}{I\+Pv4} \hyperlink{structip_1_1Datagram}{Datagram} header. 

\begin{DoxyAuthor}{Author}
Laouen Louan Mayal Belloli 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
14 May 2017
\end{DoxyDate}
An \hyperlink{structip_1_1Header}{ip\+::\+Header} has the next fields\+:
\begin{DoxyEnumerate}
\item vide\+: unsigned short that stores the version,I\+HL,D\+S\+CP and E\+CN.
\item total\+\_\+length\+: unsigned short that stores the total \hyperlink{structip_1_1Datagram}{ip\+::\+Datagram} length.
\item identification\+: unsigned short that stores the identification.
\item ff\+: unsigned short that stores the flags and fragment\+\_\+offset.
\item ttlp\+: unsigned short that stores the T\+TL and the Protocol.
\item header\+\_\+checksum\+: unsigned short that stores the header checksum.
\item src\+\_\+ip\+: \hyperlink{structIPv4}{I\+Pv4} that stores the src IP.
\item dest\+\_\+ip\+: \hyperlink{structIPv4}{I\+Pv4} that stores the dest IP. 
\end{DoxyEnumerate}
