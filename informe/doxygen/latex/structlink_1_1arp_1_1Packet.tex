\subsection{Detailed Description}
This struct implements an A\+RP packet to send a query through the subnet in order to get the response with the \hyperlink{structMAC}{M\+AC} address of the node with asigned \hyperlink{structIPv4}{I\+Pv4} ip.

\end{DoxyAuthor}
\begin{DoxyDate}{Date}

\end{DoxyDate}
The fields of a \hyperlink{structlink_1_1arp_1_1Packet}{link\+::arp\+::\+Packet} are the next\+: Hardware\+\_\+type\+: a ushort (1 for \hyperlink{structMAC}{M\+AC} address type). Protocol\+\_\+type\+: a ushort (0x0800 for \hyperlink{structIPv4}{I\+Pv4} type). H\+Len\+: a u\+\_\+char with the hardware type length. P\+Len\+: a u\+\_\+char with the protocol type length. Operation\+: a ushort (1 for query, 0 for response). Source\+\_\+\+Hardware\+\_\+\+Address\+: The source hot \hyperlink{structMAC}{M\+AC}. Source\+\_\+\+Protocol\+\_\+\+Address\+: The source hot \hyperlink{structIPv4}{I\+Pv4}. Target\+\_\+\+Hardware\+\_\+\+Address\+: The target host \hyperlink{structMAC}{M\+AC}. Target\+\_\+\+Protocol\+\_\+\+Address\+: The target host \hyperlink{structIPv4}{I\+Pv4}.
