Detailed Description

template<class INPUT>
class Parser< INPUT >

A Parser that reads inputs of type INPUT from files, construct the corresponding INPUT instance and returns it.

Author
Laouen Louan Mayal Belloli
Date
14 May 2017

The Parsers uses the >> operator and the INPUT type must implement this operator, otherwise it fails in compile time. The Parser reads one input for each line of the file, thus, the >> operator must parse a valid instance from a single line, otherwise it fails in runtime. The file can include at the begining of each line a double presition floating number and ask the parser to read that first number as a simulation virtual time to asociate with the input.

Template Parameters
INPUTData type of the input to parse from the file.
