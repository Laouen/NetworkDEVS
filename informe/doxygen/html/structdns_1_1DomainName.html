Detailed Description

This struct handles domain names with the format 'zone1.zone2.zone3...zoneN'.

Author
Laouen Louan Mayal Belloli
Date
14 May 2017

This method allows to parse domain names from strings. An example of a dns::DomainName is 'networks.devs.com' where 'networks' es zone1 'devs' is zone2 and 'com' is zone3.

dns::DomainName::DomainName ( const char *  data)
inline

Constructor from const char *.

This constructor uses a char * representation to initialize the new instance. the format must be the next:

0xlen1zone10xlen2zone20xlen3...0xlenNzoneN0x00 where there is no space between the len and the zones name. The len are unsigned chars with the length in bytes of the zones names in text format and the len = 0x00 represent the end of the domain name.

example 0x08networks0x04devs0x03com0x00 = networks.devs.com
