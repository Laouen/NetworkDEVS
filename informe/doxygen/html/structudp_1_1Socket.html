Detailed Description

This struct handles a socket states.

Author
Laouen Louan Mayal Belloli
Date
14 May 2017

If the socket is connected to a remoto host, the socket has the remote ip and port must be defined defined.

The UDP protocol uses this struct to manage all its socket states.

void udp::Socket::startReading ( )
inline

Sets the socket in reading mode to accept the next incoming udp::Segment to the socket.

If the socket is in state bound, the socket accept all the udp::Segment directed to the local ip and port. If the socket is in state connected, it accept only udp::Segment from remote socket as the one specified in the socket.
