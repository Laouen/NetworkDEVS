Detailed Description

Implement a L2 frame.

Author
Laouen Louan Mayal Belloli
Date
14 May 2017

A L2 Frame contains the next fields: preamble: an unsigned long with all the required flags. MAC_destination: The MAC Address of the destination host. MAC_source: The MAC Address of the source host. EtherType: Type of the fisical protocol (0 for Ethernet). payload: a char[1500] with the ip::Datagram or ARP Paket to send. interpacket_gat: a char[12] used to avoid solve some networks colitions problems.
