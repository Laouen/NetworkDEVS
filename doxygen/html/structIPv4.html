Detailed Description

This struct declares a IPv4 type for NetworkDEVS simulation purposes.

Author
Laouen Louan Mayal Belloli
Date
14 May 2017

It's not meant to be a full IPv4 struct, for that purpose, there is many external C++ libraries. This struct represent an IPv4 as a unsigned short array of length 4 and it was designed to easely simulate Networks using the NetworkDEVS framework.
